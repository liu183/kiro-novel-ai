## 一、时代背景：灵荒纪元

故事发生在天衍界的**灵荒纪元**——一个灵气持续枯竭、资源极度匮乏的衰退时代。

### 关键设定

- **灵气稀薄**：天地灵气只剩鼎盛时的不到一成。修士吸纳缓慢，闭关百年寸进难求。
- **资源即硬通货**：灵石、丹药、功法、法器、灵符全面短缺。**一颗中品灵石可换一户凡人三年口粮**；一炉成色尚可的疗伤丹够散修拼命。
- **功法断层**：高阶功法多在历次大劫中失传，市面流通的多是残篇、阉割版。能修到金丹者，往往因功法不全而卡死境界。
- **门派衰败**：昔日动辄数万弟子的大派，如今只剩空壳；招收弟子先看"自带多少资源"，"末等外门弟子"约等于打杂长工。
- **修为普遍偏低**：见下表"本书境界裁剪"。

### 为什么这个设定爽？

> 越稀缺，"无限供给"越炸裂。别人为一瓶下品丹拼命，主角随手批发顶阶仙丹——
> **金钱爽 / 资源碾压爽**在灵荒纪元被放大到极致。这是本书第一生产力，也是第一爽点引擎。

## 二、境界体系（继承标准 + 本书裁剪）

沿用标准九境：**炼气 → 筑基 → 金丹 → 元婴 → 化神 → 合体 → 大乘 → 渡劫 → 飞升**。

但因身处灵荒纪元，**整体修为水位大幅下移**，境界的"社会含义"被重新标定：

| 境界 | 标准时代地位 | 灵荒纪元地位（本书） |
|---|---|---|
| 炼气 | 入门凡人级 | 绝大多数修士的终点 |
| 筑基 | 修士起点 | **一城一镇的强者**，散修眼里的大人物 |
| 金丹 | 一方强者 | **一方霸主 / 大势力长老级**，巨头 |
| 元婴 | 跨域级 | **传说级**，整片区域屈指可数，多在大势力顶层 |
| 化神及以上 | 域主级 | **近乎神话**，世人只闻其名（与暗线相关） |

### 本书突破纪律（红线）

| 卷 | 主角境界目标 | 备注 |
|---|---|---|
| 第一卷 灵荒立府 | 炼气 → 筑基后期 | 主角战力靠**资源 + 法器 + 灵符 + 道侣双修**堆叠，境界本身不冒进 |
| 第二卷 | 筑基 → 金丹 | |
| 第三卷 | 金丹 → 元婴 | |
| 终卷 | 元婴 → 渡劫飞升 | 拉长每境停留，越后期越慢 |

> ⚠️ **境界 ≠ 战力**：主角是"用资源 / 装备 / 人脉碾压境界"的典型。允许越级，但越级靠的是**砸资源**，而非主角天赋爆表——这正是本书特色，务必守住。

## 三、核心金手指：万宝楼系统

主角江晏觉醒的系统，识海中呈现为一座**九层楼阁的虚影**——【万宝楼】。

### 3.1 是什么

一个"消费返利、以金币购万物"的兑换系统。万宝楼九层货架能提供：
- **功法**（炼气到飞升各阶，含已失传的完整版）
- **仙丹 / 灵药**（疗伤、突破、解毒、延寿、洗髓）
- **法器 / 灵宝**（攻防、飞行、储物、阵盘）
- **灵符**（一次性的"高境界战力罐头"：困敌、遁走、护体、攻击）
- **杂项**（灵田种子、灵兽蛋、图纸、情报、产业地契）

> 💡 **金币能买"市面上一切能想到的东西"**——不止万宝楼货架，凡是市场上有价的（灵石、铺面、消息、人情、甚至别家拍卖行的标的），金币都能拿下。万宝楼只是"总入口"。

### 3.2 灵魂机制：金币 + 消费返利（万倍返还）

**这是本书的核心爽点引擎。** 万宝楼不收灵石，只认【金币】。

- **金币怎么来 —— 靠"花"，不靠"攒"**：江晏每一笔**有效消费**，系统都按倍率**返还金币**，基础高达 **万倍返还**。即"越敢花，回得越多"，把"败家"变成滚雪球式暴富。
- **倍率随道侣数量上浮（女主越多越高）**：返还倍率 = 基础万倍 × **道侣加成**。与江晏**两情相悦、自愿结契**的道侣（即真正双修过的红颜）越多、道心契合越深，加成越高。**红颜阵容，就是他的"返利倍率"，也是他富可敌国的根。**
- **返利门槛（关键限制）**：返还**只对"为道侣 / 与道侣相关"的消费生效**。
  - ✅ 为道侣买功法、丹药、法器，或与道侣共同经营、共修所需的消费 → 触发万倍返还。
  - ❌ **给"非双修过的人"消费 → 无返还**（纯支出、有去无回）。给外人、给普通宗门、给市场散户花钱，系统一分不返。
- **起步**：系统给一笔启动金币 + 新手礼包，让"消费→返还→再消费"的循环先转起来；初期金币紧、道侣少、倍率低，制造资源管理张力。

> 设计意图：把"首富"做成**机制必然**——他想暴富，就得敢花，且必须真正与红颜结成两情相悦的道侣（杜绝胁迫/交易，见 3.3）。这既是后宫爽文的爽点引擎，也用"门槛+代价"防止系统无脑无敌。

### 3.3 双修 / 道侣机制（尺度红线见 BOOK_GUIDE）

- 本世界"双修"= **道侣共修**：两人灵气共鸣、互补阴阳、加速温养经脉与突破，是修行正法之一，**非香艳卖点**。
- 仙子们求双修，本质是求"与万宝楼之主结道缘"——既得顶级资源、又得共修助力、又能借他暴富。
- **系统铁律（人设护城河，写进机制）**：道侣契约必须**两情相悦、自愿平等**才能被系统"认可"并计入返利倍率；任何**以资源胁迫 / 利益收买 / 单方面索取**——结契无效、倍率不增、甚至触发反噬倒扣。**即：靠权钱"买"不来道侣加成，只有真心能。**
- 写法上**点到为止、以情感与境界突破为落点**，不写露骨内容（详见 `BOOK_GUIDE.md` 内容尺度红线）。

### 3.4 "返"与"散"的悖论（接全书母题）

- 道侣返利是一台印钞机 → 江晏**花得起**；但**对苍生 / 外人的散财消费没有系统返还**，是纯流出。
- 他却依然大手笔普惠散修、救济流民——系统不返他金币，但**人心、口碑、势力、情报**会"还"他（这是另一种"返还"）。
- **设计意图**：避免"为返利只对自己人好"的油腻感——他用道侣返利赚来的金币去反哺天下，正面对冲"囤积制造稀缺"的黑手（"散 vs 囤"母题）。
- 主角心法台词雏形："系统返我金币的，是她们；可我把金币散给天下——这账，时代会还我。"

### 3.5 隐藏机制（暗线 1）

- **花得越狠、返得越多，"债 / 代价"累积越快**：兑换 / 大额返还时，系统偶发**乱码 / 透支预警**（如：`万倍核销中…债主：未知`、`债务计息中…`）。
- 万倍返还并非凭空而来——它的"货源""债主""代价"是贯穿全书的核心谜题。
- 第一卷只埋设，不解释。

## 四、势力格局（第一卷范围）

> 完整势力学参考题材层 [`修仙势力类型`](../../../genres/xianxia-修仙/worldbuilding/修仙势力类型.md)。本书第一卷聚焦下列：

| 势力 | 定位 | 与主角关系 |
|---|---|---|
| **青岚宗** | 主角入门的没落中小宗门，靠收"资源弟子"续命 | 主角起点；后被其反向"收购"/改造 |
| **万宝商盟** | 垄断丹药 / 法器 / 功法买卖的庞大财阀，灵荒纪元真正的"地下皇帝" | **第一卷主要矛盾**：主角散财=砸他们饭碗 |
| **天工拍卖行** | 万宝商盟旗下，区域最大拍卖行 | 卷末高潮舞台 |
| **散修 / 流民** | 灵荒纪元的最底层，最大的"口碑 / 人心来源" | 主角的群众基础与口碑发源地（散财普惠的对象） |
| **百花谷 / 药王脉等女修势力** | 因资源短缺而求存的中小女修门派 | 红颜来源之一；求资源求合作 |
| **神秘存在** | 暗中抽取囤积灵气者（暗线 2 幕后） | 卷末露头，不揭底 |

## 五、三条暗线（一句话，详见总纲）

1. **万宝楼的代价**：系统不是白给的，货源与债主成谜。
2. **灵荒的真相**：灵气枯竭是人祸——有人在抽取、囤积整个纪元的灵气。
3. **星图碎片**：上门红颜各持一片"星图碎片"，合则可逆转灵荒，她们是天选之人。

> 三线交汇点：**资源**。主角"散"，黑手"囤"；女主们既是受害者，也是解局的钥匙。
