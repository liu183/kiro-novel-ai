# 主角人设卡 · 江晏

## 一句话

破产电商创业者猝死穿越成仙门末等弟子，觉醒"靠施恩结缘换万物"的万宝楼系统，
在灵气枯竭的灵荒纪元靠"散财行善"闷声暴富、顺手称尊。

---

## 基础信息

- **姓名**：江晏
- **性别**：男
- **外表年龄**：约 24 岁
- **前世身份**：现代电商 / 创业者，做过供应链、定价、运营、品牌；公司暴雷负债，跑路途中车祸猝死
- **今世身份**：青岚宗末等外门弟子（起点）→ 散财阁主 / "散财真君" / 民间称"活财神"
- **境界轨迹**：炼气（废铁灵根判定）→ 筑基后期（第一卷，战力靠资源 / 法器 / 灵符 / 道侣堆叠）
- **金手指**：万宝楼系统（货币＝金币；靠"消费返利·万倍返还"暴富，返利倍率随两情相悦的道侣数量上浮）

## 外貌

- **身高**：180cm，因常年挨饿略显清瘦，眼神却很"活"
- **五官**：眉眼清隽，笑起来有点"奸商式"和气；算账 / 盘算时眼睛会发亮
- **眉心**：一点极淡的金芒（万宝楼掌柜印记），激活兑换时浮现
- **穿搭**：起点是洗得发白的杂役粗袍；发家后偏爱低调而料子极好的素袍（闷声发财风），从不穿金戴银——"财不外露"
- **标志动作**：
  - 下意识"心算"（用手指无声敲算盘节奏）
  - 摸下巴盘算"这桩买卖划不划算"
  - 被红颜撩到时，用"咳，谈正事"转移话题（破功标志）

## 性格三支柱

1. **精**：商人脑，凡事先算投入产出比；信息差、供需、定价、品牌玩得溜
2. **善（且算得过来的善）**：真心待人是本性；而他对苍生的散财普惠**虽不返金币**，却能换来人心、口碑、势力——"行善"与"长远的账"在他身上同向，不别扭
3. **怂中带刚**：怕麻烦、能用钱 / 资源解决绝不动手；但护着的人被欺负时，眼睛会冷下来

### 正面特质
- **现代经营脑**：把修仙界当"待开发市场"，降维打击
- **极强分寸感**：散财但不当冤大头，精准识别"值得施恩的人"
- **护短**：自己人受欺，砸锅卖铁也要找回场子
- **反油腻**：对红颜尊重、有边界感，从不仗资源压人（机制也不允许）

### 负面特质 / 烟火气
- **抠 vs 散的矛盾**：对自己抠到家（舍不得给自己买好丹），对该花的地方一掷千金
- **怕死、嘴软**：打不过先谈、先跑、先撒符
- **路径依赖**：太爱用"砸资源"解决问题，偶尔被人将计就计（埋隐患）
- **对感情迟钝又心虚**：被直球撩到招架不住，嘴上"谈正事"，耳朵已经红了

### 人物张力（矛盾点）
- **精于算计 vs 真心行善**：他算的是"长远人心账"，恰好和善良同向
- **想低调发财 vs 越散越出名**：越想闷声，名声越响，被迫站到台前
- **靠资源碾压 vs 怕被资源反噬**：透支预警 / 债主之谜，逼他不能无限挥霍

## 背景故事（含暗线）

### 前世：暴雷的创业者
- 做电商 / 供应链起家，一度风光，因盲目扩张资金链断裂、负债累累。
- 跑路躲债途中车祸猝死。死前念头："要是当初稳一点、对兄弟厚道一点……"
- **暗线 1 伏笔**：昏迷瞬间，眼前闪过一行字 `欢迎签约·万宝楼掌柜（候补）`。

### 今世：谷底觉醒
- 穿成同名同姓的青岚宗末等外门弟子，灵根被判"废铁级"，交不起灵石份子被磋磨。
- 谷底夜觉醒万宝楼：识海九层楼阁、金币货币、"消费返利·万倍返还"的核心循环，以及"胁迫/收买结不成道侣、倍率不增"的铁律。
- 起步困境：① 穷到几乎没有启动金币，循环转不起来；② 初始无道侣，返利倍率压在最低档——他必须先"敢花第一笔"，再靠真心结下第一个道侣，才能让雪球滚起来。

### 第一桶金与人设定调
- 雨夜不求回报、几乎掏空新手礼包救下垂死的云栀——**这一步不为返利**（彼时她还非道侣，消费无返还），纯粹是"做人"。
- 也正因这份不挟恩、不交易的真心，日后才换来云栀别扭却真挚的羁绊（道侣线起点），系统返利倍率随之打开。
- 由此顿悟两条铁律：**"敢花才能富"**（消费返利）＋ **"真心才结得了道侣、堆得起倍率"**（买不来的加成）。
- 同时确立打法：**用道侣返利赚来的金币，去普惠散财、把人心做成资产**。

## 核心金手指：万宝楼系统

### 能提供
- 功法 / 仙丹灵药 / 法器灵宝 / 灵符（战力罐头）/ 灵田种子 / 灵兽蛋 / 图纸 / 情报

### 货币与核心循环：金币 · 消费返利（万倍返还）
- **金币**：万宝楼唯一货币，可买市面上一切能想到的东西（功法/丹/法器/符/灵石/铺面/情报/拍卖标的…）。
- **消费返利**：每一笔**有效消费**，系统按倍率返还金币，基础 **万倍**——"越敢花，回得越多"。
- **倍率随道侣上浮（女主越多越高）**：返还倍率 = 基础万倍 × 道侣加成；两情相悦结契的道侣越多、道心越契合，倍率越高。
- **返利门槛**：只对"为道侣 / 与道侣相关"的消费返还；**给非双修过的外人消费 → 无返还（纯支出）**。

### 双修 / 道侣（点到为止）
- 道侣共修＝灵气共鸣、助力突破，落点在"情感 + 境界"，非香艳卖点。
- 系统铁律：道缘须两情相悦、自愿平等方能被认可并计入倍率；**胁迫 / 收买 / 索取 → 结契无效、倍率不增、甚至倒扣反噬**——买不来的加成。

### 隐藏机制（暗线 1）
- 大额消费 / 返还时弹"透支预警""万倍核销""债务计息"等乱码。
- 花得越狠、返得越多，"债 / 代价"累积越快；楼阁九层逐层点亮，越高层异象越多——代价之谜贯穿全书。

## 现代经营脑（核心爽点武器）

1. **重构供需**：灵荒纪元人人囤货抬价，他反向"普惠放量"，用规模与口碑碾压
2. **定价与品牌**：明码标价、童叟无欺、"散财阁永远有货"——做成信用招牌
3. **信息差与阳谋**：囤居奇货、反向收购、舆论造势，商战降维打击
4. **流程化生产**：配比 / 工序思维 + 苓苓催熟 + 系统丹方，成丹率药效碾压同行
5. **人心即资产**：把"散财普惠"当长期投资——系统不返金币，但口碑 / 人心 / 势力滚雪球，越散越强

---

## 战斗风格：能不打就不打，要打就砸资源

- **战前**：先评估、先谈、先算账；打得过让燕惊鸿上，自己掠阵
- **战中**：灵符战力罐头 + 道侣双修临时增益 + 地形阳谋；嘴炮拖延
- **越级**：靠"砸金币兑顶阶资源 / 砸符 + 道侣共修增益"打越级，但必有代价（金币见底 / 透支预警 / 倍率不够 / 险些露馅）
- **战后**："呼，这一仗烧了多少金币……返利够不够回本？"找系统算账

## 对暧昧的态度（成长三阶段）

### 第一阶段：手忙脚乱（卷一前段）
- 被燕惊鸿直球撩、被云栀别扭关心，统统"咳，谈正事"
- 内心 OS："我前世忙到没谈过恋爱，这群仙子怎么回事？"

### 第二阶段：边界感里的心动（卷一中后段）
- 学会分辨"她们是真心，不是图我资源"，开始正视情感
- 坚持"自愿平等"原则，绝不仗资源压人（人设护城河）

### 第三阶段：护短即深情（卷末及以后）
- 红颜受威胁时第一个站出来；"我的人，轮不到你们动"
- 意识到她们身上的秘密（星图）后，选择"先护人、再解谜"
