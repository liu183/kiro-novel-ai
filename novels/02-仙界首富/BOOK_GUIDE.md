## 风格基调

- **叙事人称**：第三人称限知（主跟江晏视角，必要时切红颜 / 反派）
- **语言风格**：仙侠古风为底 + 现代经营 / 商业 / 网络梗（仅主角心理活动与系统提示用现代语）
- **爽感引擎**：以"资源碾压 + 经营建势"为主线快感，战斗为辅
- **吐槽密度**：每 3-5 章一次系统吐槽 / 前世商战梗
- **段落长度**：单段 ≤ 4 行，对话独占一行
- **章末**：必留钩子（参见 [`章末钩子`](../../universal/techniques/章末钩子.md)）

### 轮换规则
- 每 3-5 章一个明显爽点
- 不同章主用不同类型，副爽可叠加（如金钱+打脸常配对）
- 卷末用强组合（金钱+智斗+打脸+暧昧+暗线推进）

### 红线
- ❌ 连续 3 章同类型爽点
- ❌ 连续 3 章纯战斗
- ❌ 连续 5 章无爽点

## 系统机制速查（万宝楼·万倍返利）

> 写正文前必读。完整设定见 [`worldbuilding/00-overview.md`](worldbuilding/00-overview.md) 第三节。

| 要点 | 规则 |
|---|---|
| 货币 | **金币**；可购买市面上一切能想到的东西（功法/丹/法器/符/灵石/铺面/情报/拍卖标的…） |
| 赚钱方式 | **消费返利**：有效消费 → 系统返还，基础 **万倍返还**（"越敢花，回得越多"） |
| 倍率来源 | 返还倍率 = 基础万倍 × **道侣加成**；**与主角两情相悦结契的道侣越多、道心越契合，倍率越高**（女主越多越高） |
| 返利门槛 | **只对"为道侣 / 与道侣相关"的消费返还**；给"非双修过的人 / 外人"消费 → **无返还（纯支出）** |
| 结契铁律 | 道侣须**两情相悦、自愿平等**才被系统认可计入倍率；**胁迫 / 收买 / 索取 → 结契无效、倍率不增、甚至倒扣反噬**（机制层面杜绝"花钱买后宫"） |
| 代价（暗线1） | 花得越狠、返得越多，"债 / 代价"累积越快（透支预警 / 万倍核销 / 债主未知） |
| 母题接口 | 对苍生散财**无系统返利**，是纯流出；但换来口碑/势力/人心——以"散"对冲黑手的"囤" |

### 用法纪律
- 每笔关键消费要让读者看清"花了多少、返了多少、为什么能返"，让金币有重量感。
- 倍率提升要与"道侣关系的真实进展"绑定，**不能凭空跳数字**；新道侣加入＝战力/财力跃升的爽点节点。
- 初期金币紧、道侣少、倍率低 → 制造"该不该花这笔"的资源管理张力。

## 后宫 / 双修 写作纲领（重要）

本书是后宫爽文，但走"**被真心吸引 / 各有所图却各有真情**"的高级路线，不走油腻剥削：

1. **女主有独立 agency**：每位红颜都有自己的目标、能力、命运线（星图碎片），**不是依附主角的花瓶**，更不是"返利倍率的工具人"。她们靠近主角是"主动选择"。
2. **机制反油腻（关键）**：返利倍率只认**两情相悦、自愿平等**的道侣——**胁迫 / 收买 / 交易一律无效甚至倒扣**。"想靠花钱买后宫攒倍率"在本书机制上**行不通**，从根上杜绝"霸总剥削 / 物化女性"味。倍率高，是因为"她们真心选择了他"，不是"他买下了她们"。
3. **双修=道侣共修**：以"灵气共鸣 / 助力突破 / 情感升温"为落点，**点到为止**。
4. **吃醋 / 群像戏**：红颜之间是"塑料姐妹 → 真战友"的成长，有醋意有打趣，但不雌竞撕逼降智。

## 本书专属红线

- ❌ **内容尺度**：双修 / 暧昧一律**非露骨**，以情感与突破为落点，香艳处留白、转场（"翌日……"），不写性描写。
- ❌ **女主不得工具化 / 物化**：禁止"用资源换身体"式桥段；红颜接近主角须出于自愿与真情。
- ❌ 主角圣母 / 优柔（他可以善良但精明，散财是手段也是性情，绝不当冤大头）。
- ❌ 主角无脑装逼无代价（砸金币也要有"差点露馅 / 金币见底 / 透支预警 / 倍率不够"的紧张感）。
- ❌ 第一卷主角境界超过筑基后期（战力靠资源堆，境界守纪律）。
- ❌ 返利"无限白嫖"——必须有消费成本、门槛（只对道侣相关消费返还）与代价（债务累积），制造资源管理张力。
- ❌ **靠胁迫 / 花钱"买"道侣攒倍率**——机制无效，且与"女主有 agency"红线冲突，严禁这样写。
- ❌ 暗线 30 章不动。
