## 风格基调

- **叙事人称**：第三人称限知（主跟江晏视角，必要时切红颜 / 反派）
- **语言风格**：仙侠古风为底 + 现代经营 / 商业 / 网络梗（仅主角心理活动与系统提示用现代语）
- **爽感引擎**：以"资源碾压 + 经营建势"为主线快感，战斗为辅
- **吐槽密度**：每 3-5 章一次系统吐槽 / 前世商战梗
- **段落长度**：单段 ≤ 4 行，对话独占一行
- **章末**：必留钩子（参见 [`章末钩子`](../../universal/techniques/章末钩子.md)）

### 轮换规则
- 每 3-5 章一个明显爽点
- 不同章主用不同类型，副爽可叠加（如金钱+打脸常配对）
- 卷末用强组合（金钱+智斗+打脸+暧昧+暗线推进）

### 红线
- ❌ 连续 3 章同类型爽点
- ❌ 连续 3 章纯战斗
- ❌ 连续 5 章无爽点

## 后宫 / 双修 写作纲领（重要）

本书是后宫爽文，但走"**被真心吸引 / 各有所图却各有真情**"的高级路线，不走油腻剥削：

1. **女主有独立 agency**：每位红颜都有自己的目标、能力、命运线（星图碎片），**不是依附主角的花瓶**。她们靠近主角是"主动选择"，不是被资源胁迫。
2. **机制反油腻**：万宝楼缘点机制决定——**胁迫 / 交易 / 占便宜一律倒扣**。主角想强必须真心待人，从根上杜绝"霸总剥削"味。
3. **双修=道侣共修**：以"灵气共鸣 / 助力突破 / 情感升温"为落点，**点到为止**。
4. **吃醋 / 群像戏**：红颜之间是"塑料姐妹 → 真战友"的成长，有醋意有打趣，但不雌竞撕逼降智。

## 暗线设计

| # | 暗线 | 埋设手法 | 揭秘节点 |
|---|---|---|---|
| 1 | 万宝楼来历与代价 | 兑换顶阶时系统乱码 / 透支预警（如 `库存核销中…债主：未知`）；楼阁层数异动 | 第二卷起：货源 / 债主浮现 |
| 2 | 灵荒纪元真相 | 灵气抽取痕迹 / 财阀反常巨量囤货 / 古碑残文 | 第一卷末露头，第二卷揭"人祸" |
| 3 | 红颜星图碎片 | 云栀废脉残纹 / 苓苓本源印记 / 双修时浮现星图 | 第二卷揭秘起步：合则解灵荒 |

### 埋设节奏
- 每章潜伏 0-1 处细节
- 每 ~20 章 1 次显性推进
- 第一卷只埋设，不揭秘；卷末必有 1 次大埋设
- 详细技法：[`../../universal/techniques/暗线埋设.md`](../../universal/techniques/暗线埋设.md)

## 本书专属红线

- ❌ **内容尺度**：双修 / 暧昧一律**非露骨**，以情感与突破为落点，香艳处留白、转场（"翌日……"），不写性描写。
- ❌ **女主不得工具化 / 物化**：禁止"用资源换身体"式桥段；红颜接近主角须出于自愿与真情。
- ❌ 主角圣母 / 优柔（他可以善良但精明，散财是手段也是性情，绝不当冤大头）。
- ❌ 主角无脑装逼无代价（砸资源也要有"差点露馅 / 缘点见底 / 透支预警"的紧张感）。
- ❌ 第一卷主角境界超过筑基后期（战力靠资源堆，境界守纪律）。
- ❌ 缘点"无限白嫖"——必须有获取成本与消耗压力，制造资源管理张力。
- ❌ 暗线 30 章不动。

## 写作前 / 写作后检查

- [通用每章自检清单](../../universal/checklists/per-chapter-checklist.md)（必查）

### 本书附加检查项

- [ ] 出场红颜的暧昧风格是否符合差异化表？有没有串味？
- [ ] 本章是否体现"缘点的获取 / 消耗"，让资源有重量感？
- [ ] 暧昧 / 双修是否守住非露骨红线，落点在情感或突破？
- [ ] 暗线是否在按节奏推进？
- [ ] 主角是否保留"精明 + 善良 + 现代经营脑 + 怕麻烦"的至少一项特征？
- [ ] 这章的爽点是否避免了与前两章同类型？
