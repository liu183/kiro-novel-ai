# 第一卷 · 灵荒立府（章 1-100）

> 主角境界：炼气 → 筑基后期 ｜ 收红颜：4（云栀 / 苓苓 / 燕惊鸿 / 温知秋）
> 卷主线：穿越觉醒万宝楼 → 收留落难圣女立府 → 散财行善攒缘点 → 经营商战扬名 → 卷末拍卖会击退万宝商盟，"仙界首富 / 一方之尊"招牌打响。

---

## 一、本卷一句话

一个穷得叮当响的末等外门弟子，靠"对人好就能换神物"的系统，在饿殍遍野的灵荒纪元
把日子越过越富，把人心越攒越旺，最后在区域第一拍卖会上，当着满座巨头的面，
把垄断天下资源的万宝商盟，活活"散"破了产。

## 二、四幕结构

### 第一幕：穿越·觉醒·第一桶金（ch1-25）

**目标**：立人设、立金手指、收第一个红颜、第一次资源打脸。

- **ch1-5 穿越与谷底**
  - 江晏（破产电商创业者，欠债跑路途中车祸猝死）穿越成青岚宗末等外门弟子"江晏"，同名同姓。
  - 原主因交不起"灵石份子"被外门管事磋磨，灵根被判"废铁级"，吃了上顿没下顿。
  - 谷底夜晚觉醒**万宝楼系统**：识海九层楼阁，初亮第一层；货币是【缘点】，靠施恩 / 结善缘获得，强取倒扣。
  - 初始缘点几乎为零——他穷到连"行善"的本钱都没有。**第一道难题：怎么在没钱时攒缘点？**
  - 暗线1埋：兑换界面闪过一行乱码 `掌柜契约·待激活`。

- **ch6-12 收留云栀（开局第一美女）**
  - 雨夜，重伤垂死的少女云栀坠入江晏柴房——前"云霄圣女"，灵脉被仇家废掉、被追杀，走投无路。
  - 江晏咬牙用仅有的口粮 + 系统首抽（新手礼包）的一枚下品疗伤丹救她，**不求回报**。
  - 系统暴奖：真心施恩触发"暴击"，缘点猛涨 + 解锁万宝楼第二层 + 一本完整《青木长生功》。
  - 江晏顿悟系统玩法：**善良不是负担，是生产力**。人设底色定调。
  - 云栀别扭报恩：嘴上冷硬"我欠你的会还"，偷偷帮他打扫、警戒，口是心非。
  - 暗线3埋：云栀废脉处有奇异银纹，疗伤时微光流转，系统标注 `异常·星纹残片?`

- **ch13-20 第一次资源打脸**
  - 外门管事 / 富家子弟子继续羞辱江晏"废物穷鬼"，强收他的柴房 / 逼他退宗。
  - 江晏淡定拿出系统兑换的"市面绝迹"的中品丹药 / 一张困敌灵符，反手打脸。
  - 用现代"低买高卖 + 信息差"思维，倒卖一批资源给宗内散修，赚到第一笔灵石，**口碑发酵=缘点持续进账**。
  - 确立小目标台词："这破地方，迟早被我盘下来。"

- **ch21-25 苓苓出场 + 立"小灶"**
  - 后山枯灵田里，江晏用系统灵田种子复苏一小块灵田，引出沉睡的**千年灵药化形少女苓苓**（软萌仰慕型）。
  - 苓苓认江晏为救命恩人 + 唯一的家人，黏人崇拜；她的本源能催熟灵药（经营线关键资源）。
  - 第一幕收束：江晏有了第一处"产业"（灵田）、第一个伙伴（苓苓）、第一个别扭红颜（云栀）。
  - 暗线3埋：苓苓本源印记与云栀银纹"隐隐共鸣"。

---

### 第二幕：建府·聚势·商战起手（ch26-55）

**目标**：从"个人暴发户"到"有据点的势力"；引入两位红颜；与万宝商盟第一次正面冲突。

- **ch26-35 盘下产业、挂招牌**
  - 江晏低价"收购"濒临倒闭的坊市铺面 / 半座废院，挂出招牌——**"散财阁"**（对外是丹药法器铺，对内是万宝楼前台）。
  - 用现代营销：明码标价、童叟无欺、"买丹送符"、给流民施药——既赚灵石又狂攒缘点，万宝楼连升两层。
  - 黑科技爽：用配比 / 流程思维让苓苓催熟的灵药 + 系统丹方，成丹率与药效碾压同行。

- **ch36-45 燕惊鸿登场（直球女剑修）**
  - 散修女剑修燕惊鸿为给惨死同门讨公道，硬闯欺压散修的地头蛇，重伤被江晏救济（疗伤 + 一柄系统中阶飞剑）。
  - 燕惊鸿坦荡直球："你这人有意思，要不咱俩做道侣？"把江晏撩到破功（反差爽）。
  - 她成为散财阁的"武力门面"，替江晏冲锋（躺赢爽）。双修机制首次点出（点到为止，落点在助她突破筑基）。

- **ch46-55 温知秋登场 + 初撼万宝商盟**
  - 散财阁的低价普惠砸了**万宝商盟**的垄断盘，商盟地方分舵出手打压（断货源、舆论抹黑、派人砸场）。
  - 没落商会大小姐**温知秋**找上门：她家商会也被商盟挤垮，提议"合作"——拿渠道 / 人脉换江晏的货源。商战谈判型暧昧登场。
  - 江晏 + 温知秋用阳谋反制：囤居奇货、反向收购、制造"散财阁永远有货"的口碑，第一回合智斗获胜。
  - 第二幕收束：散财阁站稳，初步势力成形；商盟记恨，矛盾升级。
  - 暗线2埋：温知秋带来的情报显示，商盟在反常地巨量"只囤不卖"某些灵气晶石。

---

### 第三幕：危机·双修·暗线浮动（ch56-80）

**目标**：把矛盾升级到"区域级"；道侣双修助战；三条暗线集中埋设 / 露头；神秘人裴渊登场留谜。

- **ch56-65 商盟金丹长老施压**
  - 万宝商盟派出区域执事长老（赫连嵘的下属）金丹修士坐镇，明抢暗夺，散财阁第一次面临"境界碾压"危机。
  - 江晏靠**灵符战力罐头 + 燕惊鸿 + 道侣双修临时增益**硬扛越级，险胜但暴露底牌（差点露馅的紧张感）。
  - 兑换高阶灵符时，系统首次弹"透支预警"——暗线1显性推进。

- **ch66-73 裴渊留谜 + 灵荒线索**
  - 神秘散人**裴渊**出现，似敌似友，点破江晏"万宝楼掌柜"身份的一角，留下谜语就走（每次只留谜不抖底）。
  - 江晏循线查到一口"枯灵古井"——地脉灵气被人为抽走的痕迹。暗线2露头。
  - 古碑残文：`非天弃世，乃世弃天`。

- **ch74-80 红颜群像 + 醋意日常 + 星图微光**
  - 治愈向日常：云栀（别扭）、苓苓（黏人）、燕惊鸿（直球）、温知秋（算计）四种风格同框，醋意 / 打趣群像戏（不雌竞降智）。
  - 一次合力布阵 / 共修中，云栀银纹与苓苓印记浮出**星图光纹**并"对接"——暗线3显性大埋设。
  - 第三幕收束：江晏意识到"她们身上有秘密，且与灵荒有关"，但选择先护好她们。

---

### 第四幕：拍卖会·扬名·立府称尊（ch81-100）

**目标**：卷末高潮总爆发——天工拍卖行大会，多爽点叠加，击退万宝商盟区域势力，"仙界首富"招牌打响，卷末甩三暗线大钩子。

- **ch81-88 备战拍卖会**
  - 万宝商盟旗下**天工拍卖行**举办区域百年大拍，借机收割天下散修、彻底挤死散财阁。
  - 江晏决定"以彼之道还施彼身"：带着万宝楼的"绝版货"高调入场，温知秋操盘、燕惊鸿压阵、云栀坐镇、苓苓供货。
  - 商盟长老赫连嵘（第一卷大反派，金丹后期，资源垄断的执事）正式露面，视江晏为必除之患。

- **ch89-96 拍卖场博弈（智斗 + 金钱 + 打脸总爆发）**
  - 拍卖场上演阳谋与资金博弈：商盟想钓鱼执法 / 哄抬绞杀，江晏用"散财"反向击穿——
    当众低价放出绝版功法 / 仙丹普惠散修，瞬间收割满场人心（缘点海啸级暴涨，万宝楼升至关键层数）。
  - 打脸高潮：被全程看轻的"穷鬼末等弟子"，被揭出正是名动一方的"散财阁主"。

- **ch97-100 决战与立府**
  - 赫连嵘恼羞撕破脸动手 → 江晏以"满场缘点灌注 + 顶阶灵符 + 道侣助战"打出越级一击，击退 / 重创其势力（保留代价：透支预警 / 缘点见底）。
  - 散修拥戴，江晏正式立"散财阁"为一方招牌，"仙界首富 / 散财真君"名号传开，称尊于一隅。
  - **卷末三连钩**：
    1. 暗线1：万宝楼大涨后弹出 `债务计息中…`，楼顶最高层透出一道注视的目光。
    2. 暗线2：赫连嵘败退前一句"你以为商盟是顶?上面……"，引出灵荒幕后。
    3. 暗线3：四女星图光纹彻底连成一角残图，指向远方——还有别的"碎片持有者"在等他。
