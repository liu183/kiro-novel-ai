# 全书总纲 · 仙界首富

> 跨卷设计文档：核心理念 + 三条暗线 + 卷间节奏。具体卷大纲请进 `volume-XX-*/`。

---

## 一、核心理念

### 一句话母题

> **在一个"囤积即权力"的稀缺时代，一个"散财即变强"的人，注定要颠覆这个时代。**

主角江晏的"散"，与幕后黑手的"囤"，是全书的核心对立。资源既是爽点引擎，
也是主线矛盾，更是三条暗线的交汇点。

### 五大设计支柱

1. **稀缺世界 × 无限供给**：灵荒纪元把"金钱爽 / 资源碾压爽"放大到极致。
2. **施恩变强的系统**：缘点机制把"行善"变成硬核成长线，杜绝油腻剥削。
3. **经营 / 商战为骨**：主角用现代供需、定价、运营、品牌思维降维打击，战斗为辅。
4. **后宫差异化 + 群像 agency**：5 种暧昧风格，女主各有命运线（星图碎片）。
5. **三暗线吊胃口**：系统代价 / 灵荒真相 / 星图之秘，层层反转。

## 二、三条暗线

### 暗线 1：万宝楼的来历与代价
- **埋设**：兑换顶阶物品时系统乱码 / 透支预警（`库存核销中…债主：未知`）；楼阁九层逐层点亮，越高层异象越多。
- **揭秘**（第二卷起）：万宝楼并非凭空给予——它从"别处"调货，欠下需要偿还的"债"；江晏是签了契约的"掌柜"，而非白嫖的幸运儿。
- **影响**：主角必须在"猛兑换爽一时"与"代价临头"之间权衡，后期甚至要直面"债主"。

### 暗线 2：灵荒纪元的真相
- **埋设**：灵气被抽取的痕迹（地脉枯井、灵气倒流）/ 万宝商盟反常的巨量囤货 / 古碑残文"非天弃世，乃世弃天"。
- **揭秘**（第一卷末露头，第二卷揭）：灵荒不是天灾，是有人在**抽取并囤积**整个纪元的灵气，制造稀缺以掌控天下——万宝商盟只是其白手套。
- **影响**：主角"散财普惠"在客观上对抗"囤积制造稀缺"，被动卷入更高层博弈。

### 暗线 3：红颜们的星图碎片
- **埋设**：云栀被废灵脉里的残纹、苓苓的本源印记、双修 / 共鸣时浮现的星图光纹；不同红颜的印记可以"对上"。
- **揭秘**（第二卷起步）：上门红颜并非偶然——她们各承一片"星图碎片"，是上一纪元布下的"火种 / 钥匙"，合则可逆转灵荒。
- **影响**：把后宫从"恋爱集邮"升格为"天选者集结"，给女主们超越爱情的使命与高光，也解释"为何偏偏是她们来找他"。

### 埋设节奏
- 每 ~20 章推进 1 次暗线；第一卷只埋设，不揭秘
- 第二卷起集中揭秘、制造反转；三线在终卷收束于"资源 / 天道"

## 四、主角成长主线（贯穿全书）

1. **求生**（第一卷前段）：穿越末等弟子，先活下去、先吃饱。
2. **求富**（第一卷）：靠系统 + 经营成为一方首富，建立"散财阁 / 万宝楼"招牌。
3. **求势**（第二卷）：从"有钱"到"有势"，势力 + 道侣 + 口碑织成网，抗衡财阀。
4. **求真**（第三卷）：直面系统代价与灵荒真相，从"闷声发财"被推向"时代棋手"。
5. **求道**（终卷）：以"散"之道破"囤"之局，重定义这个纪元的财富与天道。

## 五、字数 & 节奏目标

- 单章 3000-4000 字
- 平均每 3-5 章一个爽点；连续 3 章不能同类型爽点（红线）
- 每 20 章至少推进 1 次暗线（红线）
- 每卷末：强爽点组合 + 1 次暗线大埋设 / 揭秘
