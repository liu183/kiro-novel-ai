## 一句话

灵气枯竭的灵荒纪元，资源比命贵。破产电商人江晏穿越成仙门末等弟子，
觉醒"万宝楼系统"——金币消费返利（基础万倍）、道侣越多返利倍率越高，批发顶级功法仙丹法器，
在人人抠搜的世道里活成绝对首富，顺手走上为尊之路。
