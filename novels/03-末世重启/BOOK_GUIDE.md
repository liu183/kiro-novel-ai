# 末世重启 · 专属写作规范

## 核心定位

| 维度 | 设定 |
|---|---|
| 题材 | 末世 / 重生 / 多女主 / 黑暗向 |
| 主角调性 | 极端黑化无底线、杀伐果断、极致理性、抛弃一切道德 |
| 爽点内核 | 信息差碾压 + 黑化复仇 + 果断清除一切障碍 + 多女主后宫 |
| 情感基调 | 前世被背叛的刻骨恨意 + 这一世谁挡路谁死 + 对自己的女人极度占有 |

## 主角黑化风格指南

| 场景 | 主角做法 |
|---|---|
| 叛徒 | 不杀，留着当工具人榨干利用价值后公开处刑 |
| 挡路的无辜势力 | 先警告一次，不听就连根拔起，妇孺不留隐患 |
| 谈判 | 可以笑着握手，转身就下毒/埋伏 |
| 女人 | 有用的宠着，背叛的下场比男人更惨 |
| 盟友 | 随时可以卖，只要价码够 |
| 自己人受威胁 | 十倍奉还，灭人满门 |

核心逻辑：他不是疯子，是彻底抛弃道德后的极致理性——"这个世界对我没有底线，我凭什么对它有？"

## 力量体系

### 第一阶段·纯武力（D+0 到 D+90）
- 灾变后无超能力，拼武器储备、格斗技巧、战术素养
- 主角优势：前世三年生存经验 → 顶级近战+枪械+陷阱布置
- 强者标准：能单挑十个普通人就是一方霸主
- 资源为王：谁有枪有粮谁就是老大

### 第二阶段·异能萌芽（D+90 到 D+180）
- 变异区域开始出现"异能果实"——吃了觉醒随机异能
- 果实极度稀缺，初期全球可能只有几百颗
- 抢果实成为核心矛盾，纯武力大佬开始被异能者反超
- 主角靠前世记忆知道果实刷新点，提前布局垄断

### 第三阶段·异能争霸（D+180 后）
- 果实种类体系化（战斗系/辅助系/特殊系）
- 异能可进化（通过战斗、吞噬、特殊材料）
- 出现异能果实的"组合效应"——多果实融合
- 主角及核心女主团全员异能化，碾压格局

## 世界观时间线

- D-100：主角重生，世界还正常
- D-0：灾变爆发（丧尸潮+辐射带）
- D+30：政府崩溃，割据时代开始
- D+90：异能果实首次出现
- D+180：各大势力成型，异能争霸白热化
- D+365：终极真相浮现

## 女主设定原则

### 5位核心女主
1. 冰刃 - 前世女友，亏欠→占有→"你是我的，谁碰谁死"，后期异能：冰系·绝对零度
2. 毒蛛 - 生化博士，利益绑定→身体臣服→离不开他，后期异能：毒系·腐蚀万物
3. 野猫 - 仇人之妹，棋子→宠物→唯一能让他偶尔心软的人，后期异能：兽化系·猎食本能
4. 女王 - 另一方女霸主，征服→联姻→互相撕咬又离不开，后期异能：精神系·群体操控
5. 幽灵 - 疑似另一个重生者，唯一平等对手→亦敌亦友→终极CP，后期异能：空间系·规则扭曲

### 次要女主来源
- 收服敌对势力时留下的女将/女军师
- 主角势力内的女性精英（医疗组、情报组、后勤组）
- 联姻/政治交换来的其他势力之女
- 救下的幸存者中的出色女性
- 被征服者主动献上的美人

## 爽点配方（每章至少命中1个）

1. 信息差碾压（主角预知未来精准布局）
2. 杀伐震慑（果断灭杀，让所有人恐惧）
3. 女主收服/互动（新女主加入或感情推进）
4. 势力扩张（地盘/人口/资源增长）
5. 仇人打脸（前世欺负他的人这次跪下来）
6. 实力提升（武力/异能突破）

## 暗线

1. 重生不止他一人：幽灵明显知道未来，还有没有第三个重生者？
2. 异能果实的源头：谁在"种"这些果实？灾变是果实的培养皿？
3. 黑化的尽头：当他拥有一切后，空虚感——所有人怕他，没人真正懂他
4. 前世女友的记忆：冰刃偶尔会闪回前世的画面，她也在觉醒什么？
5. 末世设计者：灾变不是天灾，是某个组织的实验——主角的重生也在计划内？

## 风格红线（不能写的）

- ❌ 主角圣母心软放过敌人
- ❌ 主角犹豫不决错失机会
- ❌ 女主无脑倒贴没有价值交换
- ❌ 大段说教讲道德
- ❌ 敌人脑残无逻辑
