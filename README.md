## 🎯 核心设定（一句话版）

- **主角**：李长生 / 程序员穿越 / 懒贪贱护短型 / 表面淡定内心慌
- **暗线**：穿越身份之谜 + 大衍共生诀隐患 + 末法时代真相
- **第一卷目标**：100 章收 4 个徒弟（殷如焉 → 李清雪 → 涂山瑶 → 姜明月前置）
- **爽点哲学**：9 种爽点轮换，反差萌为王，暗线吊胃口

## 📝 文件命名约定

- 章节大纲：`chXXX-YYY.md`（XXX = 起始章号 3 位补零，YYY = 终章号）
- 章节正文：`chXXX-章节标题.md`（XXX = 章号 3 位补零）
- 角色卡：`NN-角色名.md`（NN = 出场序）
- 卷目录：`volume-NN-卷名/`

按 3 位补零是为了 200+ 章规模下文件名能正确字典序排序。
