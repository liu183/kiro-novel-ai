# 爽点菜谱：黑科技爽

> 用现代知识 / 系统能力对古人 / 修真界进行降维打击的爽感。
> 穿越流核心爽点之一。

---

## 触发条件

- 主角有现代背景（穿越 / 重生 / 系统）
- 当前世界缺乏某种知识（物理 / 化学 / 管理 / 工程）
- 出现可以应用现代知识的场景

---

## 三段式写法

### 1. 铺垫（场景需求）
- 出现一个问题，传统方法解决不了 / 解决得很笨
- 周围人按"古法"操作
- 主角内心吐槽 / 看戏

### 2. 爆发（降维打击）
- 主角祭出现代知识（物理 / 数学 / 工程 / 管理学 ……）
- 用古人能理解的话翻译现代概念
- 周围人震惊（"这是什么仙法？"）

### 3. 收尾（普及阶段）
- 教徒弟 / 解释原理
- 留一句"在我家乡这是入门知识"的反差感
- 古人重新看待主角

---

## 雷区

- ❌ 现代知识过于专业（读者看不懂）
- ❌ 解释太长（成科普文）
- ❌ 古人零阻力接受（不真实）
- ❌ 用一次就忘 → 没有延续性
- ❌ 对手也能马上学会 → 优势消失

---

## 与人设的配合

- 程序员穿越 → 算法 / 数据库 / 调试思维
- 工程师穿越 → 物理 / 工程
- 医生穿越 → 医学 / 解剖 / 配药
- 商人穿越 → 商战 / 营销 / 资本
- 历史学家穿越 → 政治 / 阴谋

主角职业要决定他擅长哪类黑科技。
