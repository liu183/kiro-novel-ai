# 爽点菜谱：金钱爽

> 暴富、捡漏、商战碾压、资源积累产生的物质爽感。
> 现代向作品（穿越 / 都市 / 系统流）尤其常用。

---

## 触发条件

- 主角缺钱 / 资源（前期）
- 出现可以"暴富"的机会
- 主角有信息差 / 黑科技 / 系统优势

---

## 三段式写法

### 1. 铺垫（落魄阶段）
- 主角穷 / 缺资源
- 对照富人的嚣张
- 制造反差预期

### 2. 爆发（暴富阶段）
- 一笔大钱 / 一件神兵 / 一个商机
- 让钱来得"看似巧合"，实际是主角眼光
- 数字要具体（"赚了 1000 万灵石"比"赚了很多"爽 10 倍）

### 3. 收尾（享受阶段）
- 主角购物 / 升级装备 / 奖励徒弟
- 对手震惊（"他怎么会有这么多钱？"）
- 留新机会的伏笔

---

## 变体

| 变体 | 特点 |
|---|---|
| 捡漏型 | 在不识货的地方捡到宝 |
| 黑科技型 | 用现代知识赚古人钱 |
| 商战型 | 智斗 + 金钱碾压 |
| 系统奖励 | 触发系统给的暴富 |
| 资源觉醒 | 先有的东西突然值钱 |
| 散财型 | 暴富后大方分给徒弟/朋友 |

---

## 雷区

- ❌ 钱多到无意义（300 章后还在为 1 万灵石努力）
- ❌ 暴富过程太离谱（无逻辑）
- ❌ 暴富无对照 → 不爽
- ❌ 数字不具体 → 不爽
- ❌ 暴富后无消费场景 → 不爽

---

## 升级路径

资源/金钱的爽点要有"通货膨胀"：
```
铜钱 → 银两 → 灵石 → 灵晶 → 仙玉 → 大道之资
```
让主角持续有"新货币层级"可追求。

---

## 与其他爽点的搭配

- **金钱爽 + 打脸爽**：富二代被主角碾压
- **金钱爽 + 黑科技爽**：靠现代知识赚钱
- **金钱爽 + 治愈爽**：暴富后给徒弟买好东西
