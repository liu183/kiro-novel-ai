# 爽点菜谱：打脸爽

> 嘲讽 / 轻视 / 看不起 → 反杀 / 揭示真相 / 实力反转 的经典爽感。

---

## 触发条件

- 对手或路人轻视主角（不知道实力）
- 主角故意低调
- 旁观者中有"识货"的人，对比效果更强

---

## 三段式写法

### 1. 铺垫（嘲讽阶段）
- 对方表现得越嚣张 / 越自信，打脸越爽
- 让对方说出"立 flag"的话："你这种垃圾还想……"
- 让旁观者一起轻视（数量级提升）

### 2. 爆发（反转阶段）
- 一击 / 一句话 / 一个动作打破认知
- 注意"展示"而非"宣告"（让画面说话）
- 旁观者表情戏不能少（震惊脸是关键调味）

### 3. 收尾（碾压阶段）
- 对方求饶 / 改口 / 不知所措
- 主角的态度一致（淡然 / 冷漠 / 冷嘲）
- 留一句"经典台词"

---

## 变体

| 变体 | 特点 |
|---|---|
| 实力打脸 | 最直接（你强我更强） |
| 身份打脸 | 揭露身份吓退（"原来他是宗主"） |
| 智力打脸 | 算计反转（你以为你算计了我） |
| 资源打脸 | 钱 / 关系 / 资源压人 |
| 时间打脸 | 之前嘲笑的话回头打脸（"看，他真的做到了"） |

---

## 雷区

- ❌ 反派智商太低（自动打脸不爽，要有"看似聪明"的对手） 
- ❌ 主角自己拼命解释（应该让别人惊呼） 
- ❌ 打脸后立即原谅（爽感打折）
- ❌ 主角主动找人打脸（变得讨厌） 
- ❌ 打脸过于频繁 → 麻木

---

## 与其他爽点的搭配

- **打脸爽 + 战斗爽**：黄金组合
- **打脸爽 + 智斗爽**：高阶变体（不动手就打脸）
- **打脸爽 + 身份揭秘**：剧情转折点
