# 写作规范与检查清单

> 每写一章前后都看一眼。这是这本书的"风味守护者"。

---

## 🎨 五大写作原则

1. **主角要有欲望** —— 贪财 / 好色 / 护短，有欲望才有行动力
2. **每个角色独一无二** —— 暧昧风格 / 台词 / 肢体语言完全不同
3. **暗线吊胃口** —— 明线爽 + 暗线悬念，双管齐下
4. **爽点要多样** —— 不能连续 3 章同类型爽点
5. **反差萌是王道** —— 表面 ↔ 内心的反差最爽

---

## 🍬 9 种爽点类型（轮换使用）

| 类型 | 说明 | 示例场景 |
|---|---|---|
| 战斗爽 | 强者镇压、扮猪吃虎 | 仙山领域秒杀来犯之敌 |
| 打脸爽 | 嘲讽 → 反杀 | 散修嘲笑李长生废柴 → 见识真实力 |
| 金钱爽 | 暴富、捡漏 | 卖系统兑换的"黑科技"赚灵石 |
| 智斗爽 | 算计、布局 | 用现代思维降维打击修士 |
| 黑科技爽 | 现代知识乱入 | 给徒弟讲牛顿三定律辅助修炼 |
| 暧昧爽 | 师徒拉扯 | 心念牵引暴露心跳 |
| 收徒爽 | 徒弟天赋觉醒、对师父归心 | 殷如焉筑基突破 |
| 躺赢爽 | 看戏、徒弟内卷替师父打架 | 三个师妹为谁伺候师父打起来 |
| 治愈爽 | 日常温馨 | 涂山瑶把零食塞师父嘴里 |

**红线**：连续 3 章不能是同一类型。

---

## ✅ 每章自检清单

写完一章后，对照检查：

- [ ] 这一章有明确的爽点吗？属于哪一类？
- [ ] 主角是主动行动还是被动应对？（被动 ≤ 30%）
- [ ] 出场徒弟的暧昧风格符合人设卡吗？没有串味？
- [ ] 暗线有推进吗？（每 ~20 章至少埋一次）
- [ ] 有反差萌吗？（主角内心吐槽 / 徒弟表里不一）
- [ ] 吐槽密度够吗？（每 5 章至少一次系统/前世段子）
- [ ] 章末钩子够强吗？读者会想点下一章吗？

---

## ✍️ 风格基调

- **叙事人称**：第三人称限知（主要跟着李长生视角，必要时切徒弟）
- **语言风格**：
  - 现代网络用语 + 仙侠古风混搭（仅主角心理活动用现代梗）
  - 对话口语化，避免文绉绉
  - 大段描写要剪，节奏优先
- **段落长度**：单段 ≤ 4 行，对话独占一行
- **章末**：留钩子（悬念 / 反差 / 出人意料）

---

## 🚫 禁忌清单

- ❌ 主角圣母 / 优柔寡断
- ❌ 徒弟暧昧风格趋同（"嘴硬+脸红"是雷区）
- ❌ 大段世界观倾倒（融进剧情）
- ❌ 战斗连续 3 章
- ❌ 主角无脑装逼无代价（要有"差点翻车"的紧张感）
- ❌ 暗线 30 章不动（读者会忘）
