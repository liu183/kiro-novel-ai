# 每章自检清单（通用）

> 写完一章后逐项对照。本书有专属附加项目时，在 `novels/<书>/BOOK_GUIDE.md` 里补充。

---

## 必查项

- [ ] **爽点**：这一章有明确的爽点吗？属于哪一类？
- [ ] **主动性**：主角是主动行动还是被动应对？（被动 ≤ 30%）
- [ ] **差异化**：出场角色的言行符合人设卡吗？没有串味？
- [ ] **暗线**：暗线有推进吗？（每 ~20 章至少埋一次）
- [ ] **反差**：有反差萌吗？（主角内心吐槽 / 角色表里不一 / 强者弱点）
- [ ] **章末钩子**：钩子够强吗？读者会想点下一章吗？
- [ ] **节奏**：单章字数 3000-4000？没有大段描写拖节奏？
- [ ] **段落**：单段 ≤ 4 行？对话独占一行？

## 红线（不允许触发）

- [ ] 连续 3 章同类型爽点 → ❌
- [ ] 连续 3 章战斗 → ❌
- [ ] 连续 30 章暗线零推进 → ❌
- [ ] 主角无脑装逼无代价 → ❌

## 风格项（按本书风格选填）

- [ ] 现代向作品：是否有现代梗/系统吐槽？
- [ ] 穿越向作品：穿越者优势是否合理使用？
- [ ] 暧昧线作品：暧昧场景是否符合该角色的"暧昧风格"？

- [ ] 知道本章承接的上一章钩子是什么、要回收哪些伏笔
