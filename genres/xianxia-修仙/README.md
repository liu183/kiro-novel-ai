# 题材：修仙

> 同题材跨书共享的资源。具体到某一本书的设定放在 `novels/<书>/`。

---

## 📂 子目录

| 目录 | 内容 |
|---|---|
| `worldbuilding/` | 题材标准设定（境界、势力、功法分类） |
| `tropes/` | 题材常见套路（收徒流、宗门流……） |
| `scenes-snippets/` | 题材常见场景的写法参考片段 |
| `beat-types/` | 题材推荐爽点列表（薄包装，详细写法在 `analysis/beat-cookbook/by-genre/xianxia-修仙/`） |

---

## 🚀 修仙书启动检查表

新书是修仙题材时，先确认：

- [ ] 境界体系：用 [`worldbuilding/境界体系-标准.md`](worldbuilding/境界体系-标准.md) 还是自定义？
- [ ] 势力划分：用 [`worldbuilding/修仙势力类型.md`](worldbuilding/修仙势力类型.md) 还是自定义？
- [ ] 功法体系：参考 [`worldbuilding/功法分类.md`](worldbuilding/功法分类.md)
- [ ] 题材定位：收徒流 / 宗门流 / 散修流 / 长生流 / 仙凡对立 / ……
- [ ] 选用的题材爽点：见 [`beat-types/`](beat-types/)
