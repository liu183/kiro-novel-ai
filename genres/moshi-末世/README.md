## 🌍 末世题材通用元素

### 灾变类型
- 丧尸爆发（病毒/寄生/辐射变异）
- 自然灾变（地震/洪水/极端气候）
- 外星入侵
- 文明崩塌（核战/AI失控/能源枯竭）
- 复合型灾变（多种叠加）

### 力量体系常见模式
- 纯武力阶段（枪械/冷兵器/格斗）
- 异能觉醒（辐射变异/药物注射/天赋觉醒）
- 进化体系（丧尸进化/人类进化/共生进化）
- 科技路线（机甲/改造人/纳米技术）

### 核心资源
- 食物和水源
- 武器弹药
- 燃料和能源
- 药品和医疗
- 安全据点/庇护所
- 异能觉醒材料（果实/晶核/药剂）

### 势力结构
- 军方残部
- 幸存者据点（民间自组织）
- 黑暗势力（匪帮/邪教/人贩）
- 科研机构（疫苗/进化研究）
- 异能者组织

## ✍️ 末世题材写作要点

### 节奏把控
- 前期（灾变初期）：生存压力为主，节奏紧凑，几乎每章都有生死危机
- 中期（割据时代）：势力博弈为主，穿插个人实力提升
- 后期（争霸/真相）：格局拉大，揭示灾变真相，终极对决

### 氛围营造
- 环境描写突出荒凉、危险、压抑感
- 人性黑暗面的展现（灾难下道德崩塌）
- 资源稀缺带来的紧迫感和选择困境
- 信任危机（背叛是常态，忠诚才珍贵）

### 常见爽点
- 物资囤积/据点建设（基建爽）
- 碾压丧尸/变异兽（战斗爽）
- 收服人才/扩张势力（养成爽）
- 信息差碾压（重生/系统带来的先知优势）
- 打脸/复仇（前世仇人这世踩）
- 异能觉醒/进化突破（升级爽）

### 风格红线（通用）
- 避免主角无脑圣母（末世设定下不合理）
- 避免资源凭空出现（需要合理来源）
- 避免敌人智商下线（末世能活下来的都不蠢）
- 避免跳过生存细节（吃喝拉撒是读者代入感来源）

- [ ] 势力格局：有哪些主要势力？主角如何从零开始？
